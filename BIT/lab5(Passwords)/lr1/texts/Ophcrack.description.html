2.4. Описание Ophcrack

Ophcrack является программой для восстановления паролей пользователей ОС Windows. Эффективное восстановление паролей основано на использовании rainbow-таблиц (дословно — радужных таблиц). Высокоэффективная реализация rainbow-таблиц осуществлена разработчиками этого метода.

Программа оснащена графическим интерфейсом GTK+ и работает под ОС Windows, Mac OS X (Intel CPU), а также под Linux.

Позволяет восстанавливать пароли пользователей по LM и NTLM хэшам. Доступны бесплатные rainbow-таблицы для цифро-буквенных LM-хэшей.

В программе предусмотрена загрузка хэшей из локальной и удалённой баз SAM, а также загрузка шифрованной базы SAM, восстановленной с раздела Windows.

Один из вариантов распространения Ophcrack — полноценный загрузочный диск (live-CD), содержащий полнофункциональную операционную систему Linux SLAX, непосредственно программу Ophcrack и rainbow-таблицы для цифро-алфавитных паролей. Особенность такого дистрибутива — возможность его загрузки на произвольном компьютере без необходимости установки и без необходимости знания пароля администратора. Соответственно с возможностью восстановления паролей пользователей, хранящихся на жёстком диске.
