3.2. Порядок работы с BruteHash

Для восстановления паролей по имеющимся хэш-функциям воспользуемся программой BruteHash. После её запуска указываем тип используемой хэш-функции (в нашем случае это SHA1),



а также файл словаря, который содержит список возможных паролей. Для чего нажимаем кнопку «Dictionnary» и в появившемся диалоговом окне выбираем требуемый файл словаря.


После выбора словаря можно приступать к попыткам восстановления паролей пользователей. Для этого в исходном диалоговом окне вставляем требуемый хэш и нажатием кнопки "Start" начинаем попытку восстановления.

Это объясняется тем, что данная программа работает только со словарём. То есть, создав файл словаря, имеющий все комбинации слов из 4-х символов, мы можем очень быстро восстановить короткие пароли пользователей и т.д. Однако формирование словарей для более длинных паролей и случайных паролей является проблематичной задачей. Следовательно, возможности данной программы сильно ограничены и не идут в никакое сравнение с программами, использующих радужные таблицы (rainbow tables).
