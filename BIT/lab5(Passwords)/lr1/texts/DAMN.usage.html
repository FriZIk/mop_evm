3.1. Порядок работы с формирователем хэшей DAMN Hash Calculator и SAMInside

Подготовим для дальнейшей работы пары "пароль - хэш функция". По значению полученных хэш-функций в последующем будем осуществлять попытки восстановления паролей.

Для формирования LM-хэшей из паролей рользователей запускаем демонстрационную версию SAMInside. Выбираем в данной программе меню "Tools", подпункт "LM/NT-hash generator":


В появившемся диалоговом окне вводим пароль, в результате чего вычислим значение LM- и NT-хэшей для пароля «gfg3»:


В результате получим шестнадцатеричное значение LM-хэша «3B68D0B0B2F879D8AAD3B435B51404EE».

Копируем эти значения в удобное для нас место, например в "Блокнот".

Для формирования sha1-хэшей запускаем программу DAMN Hash Calculator. Вводим в ней такие-же пароли (в поле Text) и нажатием кнопки "Calculate" получаем требуемые хэш-функции:


В итоге у нас появляется список паролей и соответствующих им LM- и SHA1-хэшей. Сохраняем эти списки в текстовые файлы, например «laba_lm_hashes.txt» и «laba_sha1_hashes.txt».

Для получения возможности оценки влияния параметров паролей (длина, используемый набор символов, использование общеупотребительных слов и т.д.) на скорость и вероятность их восстановления рекомендуется выбрать пароли с различными комбинациями этих параметров.
